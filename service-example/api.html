/transaction/{id} post delete get

post

Commit. Передается ключ транзакции.

delete

Rollback. Передается ключ транзакции.

get

Ping. Должен вызываться каждые N мс. В случае задержки вызова на 2*N сервис считает соединение потерянным и происходит односторонний откат. Если сервис перестал отвечать на ping, то контроллер региструет разрыв соедениния и выполняет глобальный откат.

post /transaction/{id}

Commit. Передается ключ транзакции.

  • Request
  • Response

URI Parameters

  • id: required (string)

Headers

  • X-Transaction: (string)

    Transaction key (generated by service)

    Example:

    2cf24dba5fb0a30e26e83b2ac5b9e29e1b161e5c1fa7425e73043362938b9824

HTTP status code 200

get /transaction/{id}

Ping. Должен вызываться каждые N мс. В случае задержки вызова на 2*N сервис считает соединение потерянным и происходит односторонний откат. Если сервис перестал отвечать на ping, то контроллер региструет разрыв соедениния и выполняет глобальный откат.

  • Request
  • Response

URI Parameters

  • id: required (string)

Headers

  • X-Transaction: (string)

    Transaction key (generated by service)

    Example:

    2cf24dba5fb0a30e26e83b2ac5b9e29e1b161e5c1fa7425e73043362938b9824

Query Parameters

  • prepare: (boolean)

    Если данный параметр указан, то сервис должен проверить готовность транзакции к коммиту.

HTTP status code 200
